## Usage
* Clone this repository
    ```bash
    git clone https://github.com/rizqitaufiqf/my-ubuntu-server-setup.git
    ```
* Go to my-ubuntu-server-setup
    ```bash
    cd my-ubuntu-server-setup.git
    ```
* Chown setup.sh
    ```bash
    chown +x setup.sh
    ```
* Execute
    ```bash
    ./setup.sh 
    ```

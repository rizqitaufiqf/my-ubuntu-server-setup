## Usage
* Clone this repository
    ```bash
    git clone https://github.com/rizqitaufiqf/my-ubuntu-server-setup.git
    cd my-ubuntu-server-setup.git
    chmod +x setup.sh
    sudo ./setup.sh 
    ```
* Login Tailscale
  ```bash
  sudo tailscale up
  ```
